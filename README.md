# 👟 Syntaxwear - E-commerce de Sneakers

![Syntaxwear Logo](./images/logo/Logo.svg)

A **Syntaxwear** é uma plataforma de e-commerce moderna e elegante, especializada em tênis e sneakers de alta qualidade. Este projeto foi desenvolvido com foco em proporcionar uma experiência de compra visualmente atraente e responsiva, permitindo que os usuários explorem coleções exclusivas de diversos estilos, do casual ao futurista.

---

## 🚀 Funcionalidades

-   **Navegação Inteligente**: Menu organizado por categorias (Masculino, Feminino, Outlet) com acesso rápido a lojas e conta do usuário.
-   **Menu Mobile**: Implementação de menu responsivo utilizando a técnica de "checkbox hack" (sem dependência de JavaScript).
-   **Seção Hero Impactante**: Destaque para o produto principal com chamadas para ação (CTAs) claras.
-   **Categorização Visual**: Cards interativos para diferentes estilos de calçados (Casual, Desportivo, Moderno, Futurista).
-   **Grid de Produtos**: Layout flexível para exibição de produtos em destaque e modelos variados.
-   **Footer Completo**: Inclui formulário de newsletter, links para redes sociais e navegação detalhada do site.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas de desenvolvimento web front-end:

-   **HTML5**: Estruturação semântica de todo o conteúdo.
-   **CSS3 (Arquitetura Modular)**: 
    -   Uso de **Variáveis CSS** para fácil manutenção de cores e fontes.
    -   **Flexbox** para layouts dinâmicos e alinhamentos precisos.
    -   Organização em componentes para facilitar a escalabilidade.
-   **Design Responsivo**: Adaptado para diferentes tamanhos de tela (Desktop, Tablet e Mobile).
-   **Google Fonts**: Utilização das fontes *Ubuntu* e *Roboto Mono*.

## 📁 Estrutura do Projeto

A organização dos arquivos segue um padrão modular:

```text
Ecommerce---Sintaxwear/
├── index.html           # Página principal do site
├── css/                 # Estilos do projeto
│   ├── base.css         # Estilos globais e tipografia
│   ├── layout.css       # Estrutura de containers e grids
│   ├── reset.css        # Reset de estilos do navegador
│   └── components/      # Estilos específicos por seção
│       ├── footer.css
│       ├── header.css
│       ├── hero.css
│       ├── product-category.css
│       ├── product-grid.css
│       └── variables.css # Cores e fontes do projeto
├── images/              # Ativos visuais
│   ├── banners/         # Banners promocionais
│   ├── icons/           # Ícones de interface e redes sociais
│   ├── logo/            # Identidade visual da marca
│   └── products/        # Fotos dos tênis e modelos
└── README.md            # Documentação do projeto
```

## 💻 Como Executar o Projeto

Como este é um projeto estático (HTML e CSS), você pode visualizá-lo de forma simples:

1.  Faça o download ou clone este repositório.
2.  Navegue até a pasta do projeto.
3.  Abra o arquivo `index.html` em qualquer navegador (Chrome, Firefox, Edge, etc.).
4.  *(Opcional)* Se usar o VS Code, recomendo a extensão **Live Server** para visualizar as alterações em tempo real.
